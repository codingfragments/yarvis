Every session follows the same rhythm:

1. **Warm-up (5 min)** — A quick drill revisiting the previous session's core concept
2. **Concept & Theory ([X]–[Y] min)** — Explanation with examples and guided practice
3. **Exercises ([X]–[Y] min)** — Hands-on tasks that build muscle memory
4. **Boss Challenge ([X] min)** — A harder task that earns bonus XP
5. **Summary** — "If you remember only 3 things…" takeaway

Sessions build on each other deliberately. Don't skip ahead — later sessions assume mastery of earlier ones.
