# App Ideas

Future app/tool ideas for the Yarvis launcher. These were previously placeholder tiles.

## Files

Browse and edit files within the `~/.yarvis/` data directory and configured project paths. Tree-based file explorer with markdown preview, syntax-highlighted code viewing, and basic editing capabilities. Useful for inspecting briefings, learning courses, and settings files without leaving the app.

## Notes

A local markdown notes editor with live preview. Supports creating, organizing, and searching personal notes stored in `~/.yarvis/notes/`. Could include tagging, full-text search, and linking between notes. Aimed at quick capture during learning sessions or briefing reviews.

## Database

An interactive SQLite explorer for querying and browsing the Yarvis data store (once migrated from JSON files to SQLite). Table browser, query editor with syntax highlighting, and result table with sorting/filtering. Useful for debugging and data inspection.

## Python

An embedded Python code runner that executes scripts using the configured Python path. Includes a simple editor with syntax highlighting, output panel, and access to the `~/.yarvis/scripts/` directory. Aimed at running quick automation scripts, data transformations, or learning exercises that require Python.

## Terminal

A built-in system terminal emulator within the Yarvis window. Provides shell access (bash/zsh/fish) without switching to a separate terminal app. Useful for running commands related to learning exercises, git operations, or system administration tasks while staying in the Yarvis workflow.
